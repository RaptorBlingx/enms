🐳 QUICK DEPLOY:

1. Copy analytics folder to server:
   scp -r outputs/analytics user@server:/enms/

2. Build and start:
   cd /enms
   docker compose build analytics
   docker compose up -d analytics

3. Uncomment nginx routes in:
   /enms/nginx/conf.d/default.conf
   
4. Restart nginx:
   docker compose restart nginx

5. Test health check:
   curl http://localhost:8080/api/analytics/health

6. Access API docs:
   http://YOUR_SERVER_IP:8080/api/analytics/docs
